## 🚩 运行

1. 拉取代码
```bash
git clone https://github.com/Lukematch/admin-react.git
```
2. 安装依赖
```bash
pnpm install
```
3. 启动项目

- `dev h5`
```bash
pnpm dev:h5
```
- `dev mp-weixin`
```bash
pnpm dev:mp-weixin
```
4. 访问地址
```bash
http://localhost:6600
```
